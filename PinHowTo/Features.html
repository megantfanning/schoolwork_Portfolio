Setting Up Your Website

Pin It Button:

    Some general tips on making pins:
  • Pins should come from source images that are at least 750 pixels wide for best results. (The images don't have to appear that big on your page. They just have to be that size in the media URL used.)
  • You can choose to a make a general “pin anything” button or a pin it button specific to that image.
  • The image specific button can also show up when the user hovers their mouse over the image if you use set the hover field to true (data-pin-hover="true" ). Hover pins are turned on in this page!
  • Make sure you have concise and clever description for your pin (it helps search-ability and the user)

The Image of the cat is too small but picture of the land scape is large enough to be optimal.

To add a pin it button there are several methods the widget builder discussed above, or you can make paste the JavaScript provided by either the widget maker or the tutorial here or you can make a AJAX call which which will load asynchronously.

If you choose JavaScript remember to include

[script type="text/javascript" src="//assets.pinterest.com/js/pinit.js"][/script]

at the end of the body of your html. JavaScript tags can block rendering of the rest of the page if the network pauses so it's important to include the SCRIPT tag at the bottom of the html body rather than the header or inside a repeated template.
When creating a pin in JavaScript use www.pinterest.com/pin/create/button/ the one that appears when you use the bookmarklet is different, and using the wrong one can affect analytics reporting.

Asynchronous loading

If you are worried about the performance it makes sense to choose to have pin asynchronously load. That way if a pin hasn't fully loaded the whole page won't be waiting on the pin to load but rather it can load “in the background” while other parts of the page display. If you'd like to load the JavaScript asynchronously, you can do so by copying and pasting this snippet anywhere on your page:

[script type="text/javascript"]
 (function(d){
   var f = d.getElementsByTagName('SCRIPT')[0], p = d.createElement('SCRIPT');
   p.type = 'text/javascript';
   p.async = true;
   p.src = '//assets.pinterest.com/js/pinit.js';
   f.parentNode.insertBefore(p, f);
  }(document));
[/script]

(see developers.pinterest.com for the most up to date code or github.com/pinterest/widgets.This function can be added once and it will work for the entire page (no need to add multiple copies if you have more than one pin object).
